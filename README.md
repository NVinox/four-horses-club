# Тестовое задание: Лендинг «Клуб четырех коней»

Адаптивная верстка многостраничного лендинга, выполненная на чистых технологиях без использования сторонних библиотек и фреймворков.

## 🚀 Ссылка на проект

[Посмотреть Live Demo](https://nvinox.github.io/four-horses-club/)

## 🛠 Технологии

- **HTML5:** Семантическая разметка.
- **CSS3:** Flexbox, Grid, CSS Variables, Scroll-driven animations.
- **JavaScript (ES6+):** Модульная архитектура, ООП (классы).
- **CI/CD:** GitHub Actions (автоматический деплой).

## 💎 Особенности реализации

### 1. Архитектура JS

Проект построен на **компонентном подходе**. Для каждого сложного элемента (слайдеры, бегущая строка, плавный скролл) написан отдельный класс с методами инициализации и валидации данных.

- **Интерактивность:** Слайдеры поддерживают как автоматическое переключение, так и ручное (стрелки, пагинация-точки).
- **Масштабируемость:** Настройки компонентов передаются через `data-атрибуты` в HTML, что позволяет переиспользовать один и тот же код для разных блоков.

### 2. Оптимизация изображений

- Использование тега `<picture>` для адаптивной подгрузки форматов **WebP/AVIF** (с фоллбеком до JPG).
- Настроены приоритеты загрузки (`preload`) для критически важных ресурсов (шрифты, Hero-image).

### 3. Производительность и Адаптив

- Верстка выполнена по принципу **Mobile First**.
- Использование `ResizeObserver` для корректного пересчета геометрии слайдеров при изменении размера экрана.
- Сложные сетки реализованы на **CSS Grid** с использованием `display: contents` для сохранения семантики при перестроении блоков.
